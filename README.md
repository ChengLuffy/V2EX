## 要求

- iOS 9.0
- Swift Version 4.0
- Xcode 9.0 or later


## 运行

1. 克隆项目

```
git clone https://github.com/Joe0708/V2EX.git
```

2. 在项目根目录安装依赖库

```
 pod install 
```
3. 打开 `V2EX.xcworkspace` 运行
